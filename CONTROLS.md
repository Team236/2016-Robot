# Controls

### Left Stick
| Button | Function |
|--------|----------|
| Trigger | Shoot |
| 2 | Eject |
| 3 | Intake |
| 4 | Cock |
| 5 | Intake (override) |

### Right Stick
| Button | Function |
|--------|----------|
| 2 | Low Gear |
| 3 | High Gear |
| 4 | Invert Drive |
| 5 | Normal Drive |

### Controller
| Button | Function |
|--------|----------|
| Y | Arm up (manual) |
| A | Arm down (manual) |
| Left Stick Y | Fine arm control |
